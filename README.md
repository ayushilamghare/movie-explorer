# Movie Explorer
A React-based movie review application that allows users to browse, search, and filter movies, view detailed information, and rate them using a star-based system.

A React movie review app with TMDB live data, flip cards, dark/light themes, and user ratings.

## Features
Browse movies with essential details like title, poster, year, and genre
Search movies by title
Filter movies by genre, year, or rating
View detailed movie information including description and cast
Rate movies using a 1–5 star system
View average ratings visually
